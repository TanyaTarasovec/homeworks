Требование: Закрыть Pull Request ветки «test» в ветку «master»

Модуль: GitHub репозиторий

Имя: Закрытие Pull Request

Название: Закрытие Pull Request ветки «test» в ветку «master»

Предусловие: 
1. Открыть сайт https://github.com/
2. Авторизоваться
   Login: ***
   Password: ***
3. Перейти в раздел "Your repositories"
4. Для закрытия Pull Request необходимо заранее создать Pull Request в ветке "test"

Шаги:
1. Открыть репозиторий, в котором находится ветка "test"
2. Перейти в раздел "Pull Request"
3. В списке открыть ветку "test"
4. Проскролить вниз и выбрать способ мержа "Merge pull request"
5. Нажать на кнопку "Merge pull request"
6. Нажать на кнопку "Confirm merge"
7. Перейти  в репозиторий
8. Раскрыть ветку "master"

Ожидаемый результат:
Ветка "test" смержена в ветку "master" и отображается в ветке

Статус: Passed

Приоритет: Low

Постусловие: Закрытие браузера
